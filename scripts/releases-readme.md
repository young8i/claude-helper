# Claude 中文助手 — 公开发行仓库

> 本仓库仅存放「Claude 中文助手」的构建产物和更新清单，供用户下载和自动更新使用。

## 这是什么？

[Claude 中文助手](https://github.com/young8i/claude-helper) 是一个桌面工具，为 Claude Desktop 提供一键汉化、API 配置引导、cc-switch 安装等功能。

本仓库是**公开的发行镜像**——源码存放在私有仓库，每次发版时 CI 自动将构建产物（`.app.tar.gz` / `.msi`）推送到这里。

## 下载

前往 [Releases](https://github.com/young8i/claude-releases/releases) 页面下载最新版本。

## 自动更新

本仓库的 Releases 中的 `latest.json` 文件供应用内自动更新使用。用户无需手动访问此仓库。
